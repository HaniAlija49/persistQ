# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-07

### Added
- Initial release of MemoryHub SDK
- Complete TypeScript API client with full type definitions
- Memory CRUD operations (create, read, update, delete, list, search)
- Semantic search with vector embeddings
- API key management (generate, regenerate, get status)
- Billing and subscription management
- Dual authentication support (API key + Clerk token)
- Zero runtime dependencies (uses native fetch)
- ESM and CommonJS module support
- Full TypeScript declaration files
